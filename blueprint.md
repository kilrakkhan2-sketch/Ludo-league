# Ludo League - Color & UI Blueprint

This document outlines the complete color palette and UI component styling for the Ludo League application. The design is based on the core colors of the Ludo League logo to create a strong, premium, and royal feel.

---

## 🎨 Final Color Palette

| Role                 | Hex       | Notes                                    |
| -------------------- | --------- | ---------------------------------------- |
| Primary Background   | `#0F3D36` | Dark Royal Green - Main theme background |
| Secondary Background | `#145A4A` | For cards, sections, and popups          |
| Gold Accent          | `#D4AF37` | Main call-to-action, titles, victory     |
| Light Gold Hover     | `#E6C76A` | Hover state for gold elements            |
| White Text           | `#F5F3EE` | Main text color for readability          |
| Dark Text            | `#062420` | For use on light/gold backgrounds        |
| Border Gold          | `#C9A24D` | Borders for cards, buttons, and sections |

---

## 🧩 Component Styling Roadmap

### Header (Top Bar)

- **Background:** `Dark Royal Green (#0F3D36)`
- **Logo Text/Icon:** `Gold (#D4AF37)`
- **Menu Text:** `Off-White (#F5F3EE)`
- **Menu Hover:** `Light Gold (#E6C76A)`
- **Bottom Border:** `1px solid Gold (#C9A24D)`
- **Shadow:** Subtle shadow below the header.

### Primary Buttons (Play / Join / Start)

- **Background:** `Gold (#D4AF37)`
- **Text:** `Dark Text (#062420)`
- **Border:** `1px solid Border Gold (#C9A24D)`
- **Hover Background:** `Light Gold (#E6C76A)`
- **Hover Text:** `Dark Text (#062420)`

### Secondary Buttons (Cancel / Back / Info)

- **Background:** `Transparent`
- **Border:** `1px solid Gold (#D4AF37)`
- **Text:** `Gold (#D4AF37)`
- **Hover Background:** `Gold (#D4AF37)`
- **Hover Text:** `Dark Text (#062420)`

### Page Background

- **Gradient:** Linear gradient from `Dark Royal Green (#0F3D36)` to `Secondary Background (#145A4A)`.

### Cards / Match Panels

- **Background:** `Secondary Background (#145A4A)`

- **Title Text:** `Gold Accent (#D4AF37)`

### Notifications / Popups

- **Background:** `Secondary Background (#145A4A)`
- **Border:** `1px solid Border Gold (#C9A24D)`

### Footer

- **Links Hover:** `Light Gold (#E6C76A)`
- **Top Border:** `1px solid Border Gold (#C9A24D)`

## ✨ UI Golden Rules

1.  **Use Gold Sparingly:** Gold is the power color. Use it for important actions and highlights to maintain its premium feel.
2.  **Dark Green is the Base:** The dark green shades are the foundation of the design, creating a serious, focused gaming atmosphere.
3.  **White is for Clarity:** Use off-white for text to ensure readability against the dark backgrounds.

## 🚀 Advanced Suggestions

- **Glow Effects:** Add a subtle gold glow on hover for primary buttons.
- **Animations:** Use gold particle/spark animations for dice rolls and win screens.
- **Confetti:** Implement a gold confetti animation for match victories.
